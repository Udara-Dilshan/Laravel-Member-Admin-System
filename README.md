## 🌟 Key Features

- **🔐 Secure Authentication**: Integrated login and registration system with secure password hashing.
- **🛡️ Multi-Role System**: Separate dashboards and permissions for Admins and Members.
- **📱 OTP Security**: Extra layer of protection for user accounts via One-Time Passwords.
- **📁 Profile Management**: Advanced user profile editing with image upload functionality.
- **🔍 Smart Search & Filtering**: Efficient member directory with real-time search and filters.
- **📱 Responsive UI**: Fully optimized for mobile, tablet, and desktop views.

---

## 🛠️ Tech Stack

- **Backend:** Laravel (PHP Framework)
- **Frontend:** Blade Templating Engine, JavaScript, CSS (Bootstrap/Tailwind)
- **Database:** MySQL / SQLite
- **Tools:** Vite, Composer, Git

## ⚙️ Installation & Setup

Follow these steps to get the project running locally:

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/Udara-Dilshan/laravel-member-admin-system.git](https://github.com/Udara-Dilshan/laravel-member-admin-system.git)
   cd laravel-member-admin-system

2. **Install dependencies:**
    ```bash
    composer install
    npm install && npm run dev

3. **Environment Setup:**
   Copy .env.example to .env
   Set your database credentials in the .env file.
   Run php artisan key:generate

4. **Run Migrations:**
   ```bash
   php artisan migrate

5.**Start the server:**
```bash
    php artisan serve
```
